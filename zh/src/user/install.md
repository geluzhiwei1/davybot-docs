# 安装DaweiBot

## 使用pip安装

```bash
pip install davybot
# or
pip install -i https://pypi.org/simple/ davybot

# Start
dawei server start
# or
python -m dawei.cli.dawei server start
# Ctrl + C to exit

# Access
http://localhost:8465/app

```
